# InterKnot Android UI 规范

本规范基于 `InterKnot-Web` 移动端 UI（`MobileBottomNav.vue`、`AppHeader.vue`、`PostCard.vue`、`VirtualMasonry.vue`、`PostOverlay.vue`、`theme.css` 等）提炼，用于统一 Android 原生端的视觉与交互风格。

---

## 1. 全局调性

- 仅保留**深色模式**，不提供浅色主题。
- 所有页面背景使用 `#0a0a0a`。
- 抬升面/面板背景使用 `#1a1a1a` 或 `#222222`。

## 4. 帖子卡片（瀑布流）

- 使用 `LazyVerticalStaggeredGrid`，单列最小宽度约 160–180dp，移动端通常为 2 列。
- 列间距 14dp，卡片项间距 14dp。
- 卡片结构：
  1. 外框：背景 `#000`，圆角 24/24/0/24，padding 4dp。
  2. 内框：背景 `#222`，圆角 20/20/0/20，`clip`。
  3. 封面区：背景 `#151515`，使用后端返回的 `coverWidth / coverHeight` 计算高度；无尺寸时默认比例 **1.576**。
  4. 浏览量覆盖层：左上角，白字 14sp 700，图标 22dp，带投影。
  5. 作者行：头像压在封面下方 `-28dp`，54dp 圆形，2dp padding + `#1b1b1b` 底色；右侧作者名 15sp 700 白色。
  6. 标题：`[category] title`，默认蓝色，最多 2 行，已读变灰。

### 动效

- 封面加载：透明度 0 → 1，280ms ease。
- 卡片按压：整体 `scale(0.96)`，140ms。
- 封面按压/悬停：`scale(1.06)`，1200ms `cubic-bezier(0.22, 1, 0.36, 1)`。
- 卡片入场：`translateY(20dp) scale(0.96) → translateY(0) scale(1)`，360ms `cubic-bezier(0.22, 1, 0.36, 1)`。

## 5. 底部导航

5 个主入口，从左到右：

1. **推送/推荐**（Home）
2. **敲敲**（消息/私聊）
3. **发布委托**（Create）—— 中间凸出按钮
4. **等级**（绳网等级）
5. **我的**（Profile）

### 样式

- 背景 `#1a1a1a`，顶部 1dp 分割线 `rgba(255,255,255,0.12)`。
- 高度 58dp + `WindowInsets.navigationBars` 安全区。
- 普通项：24dp 图标 + 12sp 白色文字。
- 选中项：图标 `scale(1.2)`，300ms `cubic-bezier(0.34, 1.56, 0.64, 1)`（easeOutBack）。

### 发布按钮

- 54dp 圆形，向上凸出 `-12dp`。
- 背景径向渐变：`#ffff8c 0%` → `#fdff42 38%` → `#f4f700 66%` → `#dfe200 100%`。
- 多层阴影营造立体感：
  - 内高光 `inset 0 1.5px 1px rgba(255,255,255,0.85)`
  - 内阴影 `inset 0 -3px 6px rgba(150,152,0,0.55)`
  - 外发光 `0 10px 22px rgba(251,254,0,0.30)`
  - 接触阴影 `0 6px 14px rgba(0,0,0,0.45)`
- 外圈 `::before` 做 2600ms 呼吸光晕动画（opacity + scale）。
- 按压：`translateY(-9dp) scale(0.93)`。
- 图标：`Plus` 26dp，颜色 `#14140a`。

### Badge

- 敲敲未读 badge：红底 `#ff3838`，白字，minWidth 16dp，高度 16dp，字体 10sp/700。

## 6. 顶部 App Bar（首页）

- 背景透明或 `#0a0a0a`，滚动下滑时隐藏、上滑时显示。
- 触发阈值：滚动 delta > 6dp 且 scrollY > 80dp 隐藏；反向或回顶显示。
- 布局：
  - 左侧：登录后显示用户头像（38dp 圆形）。
  - 中部：搜索框，圆角 999dp，背景 `#161616`，高度 42dp。
  - 右侧：敲敲铃铛图标 + 未读 badge。
- 不保留桌面端的顶部 tabs，导航交由底部导航。

## 7. 分类 / Feed 标签

- 水平滚动胶囊列表，gap 8dp。
- 未选中：背景 `#222`，边框 `#222`，文字 14sp 白色。
- 选中：背景 `#BFFF09`，边框 `#BFFF09`，文字 14sp `#222`，fontWeight 700。
- 顺序：`最新` → 各分类 → 分隔线 → `关注` → `收藏`。
- 关注/收藏：未登录时点击唤起登录；再次点击已选中项切回推荐。

## 8. 帖子详情页

- 全屏页面，背景 `#0a0a0a`。
- 顶部 header：
  - 作者头像 40dp 圆形，3dp `#2d2d2d` 边框。
  - 昵称 16sp 700 白色，单行省略。
  - 等级 `Lv.N` 12sp 700 品牌色。
  - 时间 12sp `#808080`，`创建时间 · 编辑于 ...`。
  - 右上角返回/关闭按钮。
- 封面：
  - 宽度撑满，最大高度 50vh。
  - 多图时横向滑动切换，底部 dots：默认 6dp、active 18dp × 6dp，品牌色。
  - 封面计数器：右下角，背景 `rgba(0,0,0,0.65)`，白字 13sp 700。
- 标题：24sp 900 白色，可加 `[category]` 前缀。
- 正文：16sp，行高 1.7，颜色 `#e0e0e0`。
  - 链接：`#6f9cff`，下划线。
  - 引用：左边框 4dp `#BFFF09`，左 padding 16dp，颜色 `#b0b0b0`。
  - 代码块：`#1a1a1a` 背景，8dp 圆角，等宽字体。
  - 图片：最大宽度 100%，8dp 圆角。
- 底部互动栏（吸底）：
  - 评论输入框：圆角 999dp，背景 `#171717`，高度 42dp；聚焦时圆角变 20dp。
  - 左侧：`Triple（点赞/三连）`、`丁尼`、`收藏`、`更多`。
  - 图标 24dp，选中色 `#BFFF09`。
  - 评论数 badge：输入框右侧，灰色 13sp。

```
已经到底啦 [ O_X ] /
```

颜色 `#9a9a9a`，13sp。

- 指示器颜色 `#bfff09`，圆角背景 `rgba(30,30,30,0.9)`。

### 新内容提示

- 顶部居中固定，胶囊形。
- 背景 `#BFFF09`，文字 `#000`，14sp 700。
- 内容：`有 N 条新内容，点击查看`。

## 10. 动效

- 所有转场尽量使用 GPU 可合成属性：`transform`、`opacity`。
- 弹窗/页面进入：`200ms cubic-bezier(0.165, 0.84, 0.44, 1)`（easeOutQuart）。
- 弹窗/页面离开：`200ms cubic-bezier(0.895, 0.03, 0.685, 0.22)`（easeInQuart）。
- 卡片入场：`360ms cubic-bezier(0.22, 1, 0.36, 1)`。
- 按压反馈：`scale(0.96)`，`140ms ease`。
- 尊重 `android:configChanges` 与系统 `prefers-reduced-motion`，在动画设置中提供关闭选项。

---

## 11. Android 适配

- 使用 `enableEdgeToEdge()`，内容延伸至状态栏与导航栏。
- 状态栏文字颜色为浅色（深色主题）。
- 底部导航使用 `WindowInsets.navigationBars` 获得额外 padding。
- 瀑布流使用 `LazyVerticalStaggeredGrid` + `StaggeredGridCells.Adaptive(160–180dp)`。
- 图片加载使用 Coil，支持淡入动画与 GIF。
